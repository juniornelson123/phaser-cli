# Phaser CLI

  Phaser CLI is a template generator for the phaser js game framework, using Typescript (or ES6 soon).


## Instalation

  > yarn add phaser-cli or npm install phaser-cli
  

## Get Started

  > phaser new

  > cd name_project

  > yarn install or npm install
